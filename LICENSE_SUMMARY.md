## What Applies Where

### MIT License (Code)

**Applies to:**

**Location**: See [LICENSE](LICENSE) file

### CC BY-SA 4.0 (Documentation)

**Applies to:**
- Markdown files (`.md`)
- Documentation in `/docs/` directory
- README files
- Technical documentation

**Location**: See [DOCS_LICENSE.md](DOCS_LICENSE.md)

**Key Points:**
- Requires attribution
- Requires ShareAlike (derivatives must use same license)
- Allows commercial use

### Intellectual Property Notice

## Common Scenarios

### Using Code

If you use code from this repository:
1. Include the MIT License notice
2. Include the copyright notice: "Copyright (c) 2026 David Edward Sproule"
3. You can use commercially (subject to IP notice for core inventions)

### Using Documentation

If you use documentation from this repository:
1. Provide attribution to the author
2. Link to the CC BY-SA 4.0 license
3. If you modify, you must license derivatives under CC BY-SA 4.0

### Using Both

If you use both code and documentation:
1. Follow MIT requirements for code
2. Follow CC BY-SA 4.0 requirements for documentation
3. Be aware of IP notice for commercial use of core inventions
